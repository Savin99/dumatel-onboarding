    Shared Libraries¶

    orker (shared/orker)¶

    • NATS-based task worker framework
    • orker.worker — main loop с retry logic
    • orker.utils.nats.Producer/FrontProducer — NATS publish helpers
    • orker.schemas.tasks.Task — base task Pydantic model
    • Используется всеми Python-воркерами

    psdot (shared/sqlalchemy-models-and-migrations)¶

    • SQLAlchemy async models: User, Project, Chat, Message, Attachment, Schedule, Notification...
    • Alembic migrations
    • psdot.crud.* — DAL (Data Access Layer) классы
    • psdot.session_manager.SessionManager — async context manager
    • psdot.utils.init.init — инициализация БД при старте
    • psdot.utils.nats — NATS утилиты

    coral (coral/)¶

    • TypeScript monorepo (Lerna + pnpm workspaces)
    • Packages: @coral/ui — UI-компоненты (Storybook)
    • Biome 2 для линтинга
    • Nexus package registry для публикации
    • Задачи по наполнению в бэклоге (DOTMIND-1404/1405)
