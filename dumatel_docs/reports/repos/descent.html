    Descent — Yandex Disk Import Worker¶

    Назначение и роль в системе¶

    Worker импорта файлов из Yandex Disk. Получает задачи через NATS, скачивает файлы/директории из Yandex Disk, загружает их в Sprut (backend), создаёт нотификации об успехе/ошибке.

    Технологический стек¶

    Python 3.13, orker (NATS), psdot (PostgreSQL), aiohttp, Pydantic 2.12, Sentry SDK.

    Точки входа¶

    • source/src/entrypoints/main.py — NATS worker
    • docker-entrypoint-main.sh

    API / интерфейсы¶

    • NATS Consumer: nats_descent_subject
    • HTTP-клиент: Sprut API, Yandex Disk API, Yandex Download API

    Зависимости от других сервисов¶

    Sprut (HTTP), Yandex Disk (HTTP), PostgreSQL, NATS.

    Текущее состояние¶

    Активный. Последний merge: 2026-02-27. Ответственный: e.vlasov.

    Ключевые TODO / tech debt¶

    • Тесты с allow_failure: true
