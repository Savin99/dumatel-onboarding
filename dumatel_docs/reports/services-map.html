    Связи между сервисами (схема data flow)¶

    Browser
  └─► Seadragon (Next.js)
        ├─► /api/chat → SSE ← NATS JetStream ← Amigofish (AI results)
        ├─► /api/files → Sprut → krill (PDF parse) → anglerfish (index)
        └─► REST /api/* → Sprut
                ├─► PostgreSQL (asyncpg via psdot)
                ├─► Milvus (vector search)
                ├─► Keycloak (token validation)
                ├─► Doubloon (billing) → Tinkoff API
                └─► NATS (task publish)
                        ├─► Amigofish (query worker)
                        │     └─► LLM APIs (OpenAI/Claude)
                        ├─► Anglerfish (indexing worker)
                        │     ├─► LlamaIndex (doc parsing)
                        │     ├─► Milvus (store embeddings)
                        │     └─► Doubloon (billing hold)
                        ├─► Krill (PDF parser + OCR)
                        ├─► Descent (Yandex upload)
                        ├─► Gull (email + CRM)
                        │     ├─► Bitrix 24
                        │     └─► SMTP (email)
                        ├─► Bilge (subscription billing tasks)
                        ├─► Shark (scheduled tasks)
                        └─► ConvictFish (scheduled tasks)

Chronograf (scheduler)
  └─► NATS → Bilge / Shark / ConvictFish tasks

    Библиотека Используется в

    psdot (shared/sqlalchemy-models) Все Python сервисы с БД: sprut, doubloon, amigofish, anglerfish, gull
    coral (coral/) seadragon (через Nexus package registry)
