    Архитектура: Feature-Sliced Design (FSD)¶

    app/ (страницы) -> entities/ (13 доменных сущностей) -> features/ (34 модуля) -> widgets/ (7) -> shared/ (UI, API-клиенты, утилиты) -> server/ (NATS, AI SDK bridge).

    Vitest 3.2.4, Playwright 1.53.1, MSW 2.10.2, Biome 2.0 + ESLint 9.29, pnpm 9.15 + Turbopack.

    ML/AI¶

    Модели¶

    GPT-4o (основной LLM), Claude (альтернативный), gpt-4o-mini (LangGraph R&D), text-embedding-3-small (embeddings), multilingual-e5-base (классификация Sealayers).

    RAG Pipeline¶

    Индексация: документ -> chunking (LlamaIndex) -> embeddings (OpenAI) -> Milvus (hybrid BM25 + dense). Поиск: query -> hybrid search -> RRF -> 80 чанков -> reranker -> 8 в контекст LLM. OCR: marker-pdf (PyTorch, surya-ocr).

    Agent¶

    Текущий (Amigofish): orker-based, tools (RAG, Web, Content, Tree, MindMap, Artifact, Answer), Jinja2-промпты. Следующее поколение (LangGraph): multi-agent -- Coordinator -> classify -> RetrievalAgent -> AnalystAgent -> format_output.

    Databases¶

    PostgreSQL 17¶

    Модели (psdot): User, Organization, Project, Directory, Document, Chat, Message, Attachment, Balance, Tariff, Subscription, Transaction, Card, Freeze, ChargeOff, Notification, Schedule, LegalDocument. Миграции: Alembic (в psdot). DAL-классы для CRUD.

    Milvus 2.5.14¶

    Векторная БД для RAG. Коллекция на каждый проект. Hybrid search: BM25 (sparse) + dense vectors. Fusion: RRF. SDK: pymilvus (gRPC). Зависимость: etcd 3.5.18.

    Redis 8.2.1¶

    Cache через aiocache: коды верификации email (TTL), состояние AI-генерации (complete_{chat_id}).

    NATS JetStream 2.11.8¶

    Persistent messaging. Streams: COMPLETIONS (AI streaming, Max Age 5 мин). Consumer groups для горизонтального масштабирования. DeliverPolicy: New (основной), StartTime (resume). Retry: 3 NAK, dead letter queue (orker). Anglerfish: 2 реплики (index + unindex).
