    Dumatel -- API-контракты¶

    REST API (Sprut)¶

    Base URL: /api | Формат: JSON | Auth: Bearer JWT (Keycloak JWKS) | Пагинация: Cursor-based (sqlakeyset) | OpenAPI: /openapi.json

    Группы эндпоинтов (17 роутеров, 60+ эндпоинтов)¶

    Группа Prefix Описание
    Auth /api/auth Регистрация, логин, refresh, verify-email, forgot/reset-password (6)
    Document /api/document CRUD документов + индексация через NATS (5)
    Chat /api/chat Список, получение, остановка генерации, pin (4)
    Message /api/message Отправка (+ NATS complete_tasks), список (2)
    Project /api/project CRUD проектов (+ Milvus collection create/drop) (5)
    Organization /api/organization CRUD, участники, реферальные коды, блокировка (12)
    Profile /api/profile Профиль, фото, пароль, auto-renew, referral, delete (8)
    Billing /api/billing Проксирование к Doubloon: тарифы, баланс, карты (9)
    Attachment /api/attachment Загрузка, получение вложений чата (2)
    Notification /api/notification Список, прочитать одно / все (3)
    Onboarding /api/user/onboarding Welcome popup (1)
    Yandex /api/yandex OAuth, bind/unbind, дерево, импорт с Yandex Disk (5)
    Legal /api/legal Оферта, соглашение, политика, условия биллинга (4)
    Tree/Directory /api/tree, /api/directory Иерархия каталогов, CRUD директорий (6)
    Health /api/health Healthcheck (1)

    Формат ошибок¶

    { "detail": "Error description", "status_code": 400 }

    Middleware перехватывает: AuthError -> 401/403, DBError -> 500, RequestValidationError -> 422, HTTPException -> код, иначе -> 500.

    Аутентификация¶

    Клиент Механизм Заголовок
    Фронтенд JWT Keycloak Authorization: Bearer <token>
    Service-to-service Статический токен X-Friend-Token: <token>
    Комбинированный JWT или Friend-Token Оба (dependency users_and_service_header_permissions())

    Проксирование: Seadragon /api/sprut/[...path] -> извлекает Bearer из NextAuth-сессии -> инъектирует в запрос к Sprut.

    REST API (Doubloon)¶

    Auth: X-Friend-Token middleware (JWT пользователя не проверяется). 17 эндпоинтов /api/billing:

    Endpoint Method Описание
    /balance GET Баланс: quota, purchased, debt, tariff, next_billing_date
    /hold POST Резервирование кредитов -> Freeze(ACTIVE)
    /commit POST Списание -> Freeze(COMPLETED) + ChargeOff
    /release POST Отмена резерва -> Freeze(CANCELLED)
    /buy POST Покупка кредитов -> Tinkoff -> anchor task
    /change-plan POST Смена тарифа -> Tinkoff -> anchor task
    /tariffs GET Список тарифов
    /transactions GET История транзакций
    /transaction/{id}/complete POST Завершение транзакции
    /transaction/{id}/fail POST Ошибка транзакции
    /process-auto-renewals/{id} POST Автопродление -> Tinkoff -> Anchor
    /attach-card POST Привязка карты -> anchor_card task
    /save-card POST Сохранение карты (pan, rebill_id, exp_date)
    /card/make-primary PATCH Основная карта (card или SBP)
    /card/delete DELETE Удаление карты (+ Tinkoff removeCard)
    /card/list GET Список карт и SBP-аккаунтов
    /auto-renew POST Вкл/выкл автопродления

    AI Streaming¶

    Subject Producer Consumer Описание
    completion.{chat_id} Amigofish (FrontProducer) Seadragon (thinker-model.ts) AI-ответ

    Stream COMPLETIONS: Max Age 5 мин, subjects completion.*.

    SSE Streaming (Browser)¶

    Seadragon реализует LanguageModelV2 (AI SDK 5.0) поверх NATS через thinker-model.ts. Клиент использует стандартный useChat/useChatUI.

    Типы событий: text-delta, reasoning-delta, tool, citations, meta, finish, [DONE].

    Reconnect: ChatResumeHandler -> POST /api/chat/{chatId}/resume -> NATS DeliverPolicy.StartTime. Retry: 3 NAK, delay 1 сек.

    Type Generation¶

    pnpm generate-api-types -> openapi-typescript v7.8 -> shared/api/openapi/types.generated.ts. REST-клиенты в shared/api/clients/ (12 файлов: Profile, Chat, Projects, Documents, Billing, Organizations, Attachment, Notification, Onboarding, Yandex, Auth, Supplier).
